# Student Management System (Python)

## Project Overview
This is a simple **Student Management System** built using Python.
It allows users to store and manage student details using a list of tuples.

## Features
- Add new students
- Display all students
- Find the class topper
- Show pass/fail results
- Menu-driven program

##  Concepts Used
- Lists and Tuples
- Loops
- Conditional statements
- Lambda function
- User input handling

## How to Run
1. Install Python (version 3.x)
2. Save the file as `student_management.py'

## Learning Outcome
- Understand how tuples store structured data
- Learn menu-driven program design
- Practice basic data processing logic
